📡 GoatCounter 연동 (무료 Analytics)

설정 방법:
1. goatcounter.com에서 무료 계정 생성
2. 사이트 코드 선택 (예: c0mput33 → 대시보드: c0mput33.goatcounter.com)
3. Settings → API tokens → 새 토큰 발급
4. 아래에 입력 후 저장
5. 블로그 _config.yml의 analytics.goatcounter.id에 사이트 코드 추가
6. pageviews.provider: goatcounter 설정 시 각 포스트에 조회수도 표시됨
